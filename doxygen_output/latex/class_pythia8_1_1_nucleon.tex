\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_pythia8_1_1_nucleon}{Nucleon}} class represent a nucleon in a nucleus. It has an id number (proton or neutron) an impact parameter position (absolute and relative to the nucleus center), a status and a state to be defined and used by a \mbox{\hyperlink{class_pythia8_1_1_sub_collision_model}{Sub\+Collision\+Model}}.
