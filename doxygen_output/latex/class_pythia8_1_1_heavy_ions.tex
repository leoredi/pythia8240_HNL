\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_pythia8_1_1_heavy_ions}{Heavy\+Ions}} contains several standard \mbox{\hyperlink{class_pythia8_1_1_pythia}{Pythia}} objects to allow for the combination of different kinds of nucleon-\/nucleon events into a heavy ion collision event. The actual model for doing this must be implemented in a subclass overriding the the virtual\textquotesingle{}\mbox{\hyperlink{class_pythia8_1_1_heavy_ions_a68b4a2b032793a0ad6e59416c9be1cff}{init()}}\textquotesingle{} and \textquotesingle{}\mbox{\hyperlink{class_pythia8_1_1_heavy_ions_ae00de173e7462dd6f812defce5a0ffab}{next()}}\textquotesingle{} functions.
