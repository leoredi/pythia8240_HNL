\doxysubsection{Detailed Description}
This is the heavy ion user hooks class which in the future may be used inside a \mbox{\hyperlink{class_pythia8_1_1_pythia}{Pythia}} object to generate heavy ion collisons. For now it is used outside \mbox{\hyperlink{class_pythia8_1_1_pythia}{Pythia}} and requires access to a number of \mbox{\hyperlink{class_pythia8_1_1_pythia}{Pythia}} objects.
